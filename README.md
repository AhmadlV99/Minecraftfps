# MorePlusFPS v2 (Fabric)

**Minecraft:** 1.21.11+  
**Loader:** Fabric (client-side only)

## Features
- Aggressive Entity & Block Culling
- **Removes ALL particles** (huge FPS gain on low-end devices)
- No Forge / NeoForge
- Android friendly (Pojav / Mojo)

## Notes
- Client-side only
- Fabric API required
- Particles are fully disabled for maximum performance
